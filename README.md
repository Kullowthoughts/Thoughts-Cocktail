# Thoughts Cocktail

**Thoughts Cocktail** is a simple web app that allows you to explore cocktail recipes, search for drinks, and view detailed instructions — all in the browser. No server setup required.

---

## Features

* Search cocktails by name or ingredient
* View detailed recipes and instructions
* Clean, responsive design for both desktop and mobile
* Runs entirely in the browser — no Node.js or backend needed

---

## Getting Started

You can open the app directly in your browser or use **Live Server** for a better experience.

### Using Live Server (Recommended)

1. Open the project folder in **VS Code**.
2. Right-click `index.html` → Select **Open with Live Server**.
3. Your default browser will open the app with live reload enabled.

### Without Live Server

1. Open the project folder.
2. Double-click `index.html` or open it in any browser.
3. The app will run, but changes won’t automatically refresh.

## Usage

1. Open the app using Live Server or your browser.
2. Use the search bar to find cocktails by name or ingredient.
3. Click on any cocktail to view detailed instructions and recipe.
